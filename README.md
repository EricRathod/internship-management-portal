# Internship Management Portal

## Overview

The Internship Management Portal is a Python and SQLite-based application designed to manage students, companies, and internship applications. The system allows tracking of internship opportunities, application statuses, and recruitment workflows.

## Features

* Student Registration
* Company Registration
* Internship Applications
* Application Status Tracking
* Search Applications
* Analytics & Reporting

## Technologies Used

* SQLite
* Pandas
* Google Colab
* GitHub
